# heesunkiminsg.github.io

Personal homepage of Heesun Kim, built with [Jekyll](https://jekyllrb.com/) and
hosted on [GitHub Pages](https://pages.github.com/).

**Live site:** https://heesunkiminsg.github.io

## Editing

- **Bio / Projects / Contact** — edit `index.html`.
- **Blog posts** — add a Markdown file to `_posts/` named `YYYY-MM-DD-title.md`.
- **Styling** — edit `assets/css/style.css`.
- **Site title, email, links** — edit `_config.yml`.

Changes pushed to the `main` branch are published automatically within a minute or two.

## Running locally (optional)

```bash
bundle install
bundle exec jekyll serve
# open http://localhost:4000
```
