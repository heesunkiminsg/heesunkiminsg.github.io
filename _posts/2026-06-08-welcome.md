This is my first post. The site is built with [Jekyll](https://jekyllrb.com/)
and hosted for free on [GitHub Pages](https://pages.github.com/).

To write a new post, add a Markdown file to the `_posts/` folder named like
`YYYY-MM-DD-title.md`, with a small front-matter block at the top:

```yaml
---
layout: post
title: "Your title here"
date: 2026-06-08 09:00:00 +0800
---
```

Everything below the front matter is plain Markdown — headings, **bold**,
_italics_, lists, links, and code all work out of the box.
